# SniffCat

> Curriculum design of Computer Communication and Network

- 环境：python3.6
- 依赖库：
    * `scapy`
    * `wxpython`
    * `wmi`
    * `netifaces`
    * `cx_Freeze`
    * `py2app`
- 平台：Windows 平台可用，macOS 中会出现丢包 bug
- 编译方法：若将其编译为可执行文件，请在命令行下进入本目录，Windows 系统运行`python setup.py build`，macOS系统运行`python setup.py py2app`
